# TextFileSplitter

Browser tool to split large text files into smaller parts, with optional regex filtering and text replacement.

## Run
Open `index.html` in Chrome or Edge.

## Deploy to GitHub Pages
1. Create a repository named `TextFileSplitter` on GitHub.
2. Upload all project files from this folder.
3. In repository settings, open **Pages**.
4. Set source to **Deploy from a branch**, branch `main`, folder `/ (root)`.
5. Your site URL will be:
   `https://<your-github-username>.github.io/TextFileSplitter/`
